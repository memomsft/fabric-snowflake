# 3. Setup in Snowflake

Run this SQL as ACCOUNTADMIN or equivalent:

```sql
CREATE ROLE IF NOT EXISTS FABRIC_MIRRORING_ROLE;
GRANT USAGE ON DATABASE DEMO_DB TO ROLE FABRIC_MIRRORING_ROLE;
GRANT USAGE ON SCHEMA DEMO_DB.PUBLIC TO ROLE FABRIC_MIRRORING_ROLE;
GRANT USAGE ON WAREHOUSE DEMO_WH TO ROLE FABRIC_MIRRORING_ROLE;
GRANT SELECT ON ALL TABLES IN SCHEMA DEMO_DB.PUBLIC TO ROLE FABRIC_MIRRORING_ROLE;
GRANT SELECT ON FUTURE TABLES IN SCHEMA DEMO_DB.PUBLIC TO ROLE FABRIC_MIRRORING_ROLE;
GRANT CREATE STREAM ON SCHEMA DEMO_DB.PUBLIC TO ROLE FABRIC_MIRRORING_ROLE;
```
